    This is an example of JavaScript ES6, In this project ES6 features are used such as

    • String Literals - String literals in JS are written using tilda symbol instead of quotes`` anything inside tilde will be rendered as it is, we can write multi-line string as well.
    • String Template - String templates can be written as ${variables}
    • let - let is substitute for var, but let has block scope but var has function scope.
    • const - const is substitute for var, with block scope, also const is constant and once assigned cannot be reassigned.
    • forEach - Used to iterate through collection such as array
